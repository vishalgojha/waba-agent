# `waba ai` Example Prompts

## Send Template

```bash
waba ai "send followup template to Vishal at 919812345678 for ACME with params John"
```

## Send Text

```bash
waba ai "send 'Thanks for your inquiry' to 919812345678 for acme"
```

## Schedule Template

```bash
waba ai "schedule followup template for ACME tomorrow 10am to +919812345678 with params Vishal"
```

## Schedule Text

```bash
waba ai "schedule reminder for acme tomorrow 10am to +919812345678"
```

## List Templates

```bash
waba ai "list my templates for acme"
```

## Show Memory

```bash
waba ai "show memory for acme client"
```

## Webhook Setup

```bash
waba ai "setup webhook for https://my-public-domain.com"
```
