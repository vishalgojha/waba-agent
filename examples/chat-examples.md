# `waba chat` Example Sessions

## Start Chat For A Client

```bash
waba chat --client acme-realty --lang en
```

## Resume Last Session

```bash
waba chat resume --client acme-realty
```

## See Session History

```bash
waba chat history --client acme-realty
```

## Prompts You Can Use Inside Chat

- `I got 5 new leads from 99acres for ACME`
- `qualify them`
- `schedule follow-up tomorrow 10am`
- `show templates`
- `show memory`
- `Hindi mein reply karo`

## Quick Commands Inside Chat

- `/help`
- `/leads`
- `/schedule`
- `/status`
- `/lang hi`
- `/lang en`
- `/exit`
